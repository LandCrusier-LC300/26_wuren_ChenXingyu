1.论文题目：You Only Look Once: Unified, Real-Time Object Detection
2.研究背景：在yolo即单阶段检测出现前，主要为DPM和R-CNN这类双阶段检测作为图像检测的主流算法，该算法的特点是现在图像上对目标画框（DPM为滑动窗口全图像穷举搜索，R-DNN为通过卷积神经网络选择候补框），再加入分类算法进行分类。这中双阶段检测具有精度极高的优点，但是存在检测时间巨慢的缺点。在这样一种情况下，论文作者提出yolo单阶段检测算法，通过网络学习整张图片的特征，进行画框和分类同时进行的操作，实现在较高精度下提升检测速率，为自动驾驶等需要高速率检测图像的领域的发展提供了算法基础。同时，也与人“一眼识别物体与类别”的特性相契合
3.核心检测原理：提出网格划分机制：将输入图像划分为S×S网格（PASCAL VOC 数据集取 7×7），物体中心落在哪个网格，就由该网格负责预测。每个网格输出 2 组边界框（含坐标、宽高、置信度）以及 20 类目标的类别概率，最终整合为7×7×30的预测张量；明确定义置信度公式（物体存在概率 × 预测框与真实框交并比 IoU）。
4.神经网络搭建：24*卷积神经网络曾（负责图像特征提取）+2*全连接层（负责做分类任务）
5.训练方式：预训练：在 ImageNet 数据集上完成卷积层预训练，获取通用视觉特征；
迁移改造：新增卷积与全连接层，适配检测任务；
激活函数：除输出层外全部使用 Leaky ReLU，避免梯度消失；
损失函数：采用加权平方误差损失，针对坐标误差、无物体网格置信度误差设置不同权重，同时对框的宽高做开根号处理，缓解大小目标误差不均衡问题；
训练策略：配置迭代轮数、批次大小、动量、学习率衰减方案，使用 Dropout 与数据增强（缩放、色彩调整）防止过拟合。
6.数据集：现在Imagenet上进行预训练（统一224*224的尺寸），然后在专用的图像模型训练集上进行模型训练微调和测试（PASCAL VOC 2007 + VOC 2012 ，尺寸统一为448*448，来适配检测任务、获取细粒度特征）
7.个人理解：整体训练如同先使用课本进行学习，然后使用考研资料进行巩固和做题。识别过程来讲，打个不恰当的比喻：双阶段如同先大量看题，把可能考的题目筛出来一道一道去做，这样就很慢；单阶段如同在通篇扫题的同时现场就把题做完，虽然准确率没那么高，但复习起来很快
