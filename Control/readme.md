一。PID算法：
比例积分微分控制器
（1）比例P：根据差距，通过比例运算给定修正的输入值
（2）微分D：预测偏差的变化趋势，提前进行微调控制，减少振荡
（3）积分I：不论比例控制器的参数设定多么完美，在实际控制过程中必然还会存在一丁点误差，I控制器用来补齐这个误差
